# ForCookie CMP template Google Tag Manager

Bu şablon ile, web sitesi kullanıcılarından çerez onayı almak ve kullanıcının onay tercihlerine uygun olarak çerezleri ayarlayan GTM etiketlerini kontrol etmek için Google Tag Manager'da ForCookie CMP çerez onay ve rıza platformunu kullanabilirsiniz.

Lütfen https://www.forcookie.com adresinde bir hesap oluşturun ve uygulama için şu kılavuzu izleyin: https://www.forcookie.com/forcookie-tagmanager-nasil-kurulur/
